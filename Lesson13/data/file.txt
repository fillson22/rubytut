Работа с Гитом на своем компе, локально!

1. git init имя     - создает пустую папку репозитория(архива) и именем(имя)
                      если без имени то использует текущую папку
2. git status       - команда показывает что в вашей папке появилось нового или 
                      какие файлы или папки изменились
3. git add .        - гит начинает следить за всеми файлами находящимися в 
                      репозитории
4. git commit -m "Коментарий"   - делает сохранение(как в игре) файлов и папок 
                                  за которыми он следит 
5. git log          - показывает все сохранения
   git log -n       - где n количество последних коммитов которые хочешь видеть
6. git checkout номер_идентификатора    - возвращает в то положение когда был 
                                          сделан коммит

Работа с GitHub

1. git remote show  - показывает список удаленных репозиториев привязанных к 
                      текущему.
//2. ssh-keygen       - генерить ключ для ГитХаба

2. Создаем репозиторий на ГитХабе и  действуем по иструкции:
3. git remote add origin https://github.com/fillson22/first_step.git    - создает связь между локальным и удаленным репозиторием
4. git push -u origin main  - пушит на гитхаб наш проект
5. git pull         - стянуть комит с удаленного репозитория
